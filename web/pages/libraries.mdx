# JavaScript libraries

FeedrApp is consumed by a couple of public Open Source projects that load RSS feeds on page load and convert them into HTML.

- [Vanilla RSS](https://github.com/sdepold/vanilla-rss) ([NPM](https://www.npmjs.com/package/vanilla-rss/))
- [jQuery RSS](https://github.com/sdepold/jquery-rss) ([NPM](https://www.npmjs.com/package/jquery-rss/))
